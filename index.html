ttrss -- Tiny Tiny RSS Reader

ttrss is a Tiny Tiny RSS Reader App for the Nokia N9, the Jolla smart phone and mobile devices running Ubuntu, written using Qt/QML. It uses the Tiny Tiny RSS API.

If you want to contribute, feel free to use the Issue Tracker, send me a Pull Request or buy me a beer (in €):

If you want to discuss, there is the official Maemo.org Thread.

ttrss is available for download at OpenRepos.net (Harmattan/N9 and Sailfish/Jolla), so just go ahead and add my OpenRepos repository so you will get the updates straight to your phone.

If you are using this app with MeeCoLay on your N900, you need another package to support com.nokia.extras 1.1. As the app uses the ShareUI since v0.2.3, there is a special build available, which has this feature disabled: Download.

In the update to version v0.3.3 the location of the configuration file changed. To use your old configuration file, copy it to the appropriate place:

mv ~/.config/ttrss/ttrss.conf ~/.config/Hauke\ Schade/ttrss.conf
